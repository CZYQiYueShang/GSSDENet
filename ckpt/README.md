## ckpt
The initial arrangement of the `ckpt` folder is as follows:
```
ckpt/
├── pretrained/
│   ├── resnet50-deep.pth
│   ├── resnet101-deep.pth
│   ├── convnext_tiny.pth
│   ├── convnext_small.pth
│   └── convnext_base.pth
└── trained/
    ├── GSSDENet/
    │   └── ResNet101-200.pth
    └── GSSDENet-S/
        ├── ResNet101-200.pth
        └── ConvNeXtB-200.pth
```
